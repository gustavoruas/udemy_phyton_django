DOM - Document Object Module

Some important document references and their functionality for HTML page attributes

  • document.URL - This is the URL of the whole site
  • document.body - this is everything inside the body tag
  • document.head - This is everything in the header of the page
  • document.links - These are all the links on the page

Here are some methods to fetch HTML elements

  • document.getElementById(id) - returns element with the defined id
  • document.getElementsByClassName(name) - returns elements with the specific attribute name
  • document.getElementsByTagName(tag) - returnslist off all element tags defined by param
  • document.querySelector() - Returns the first object matching the CSS style selector
    • CSS Style Selectors - .(class) , #(id), tagName
  • document.querySelectorAll() - Returns all object matching the CSS style selector
