Jquery tag basics

To fetch the element by all tag name

  • $("a") - Fetches all a link tags in the document
  • $(".name_of_class") - Fetchess all tags that contain the referenced class name
  • $("#id_name") - Fetches all tags that contain the referenced ID

Change CSS of specific object

  • $("#id_name").css("property", "value") - Changes a single property of CSS to the ID of tag element selected
  • Defines multiple CSS attributes via a CSS object in JQUERY
    var cssobjc = { "color" : "blue", "border" : "20px solid green" }
    $("#id_name").css(cssobjc)
  • $("tag").addClass("className") - This adds a new class into the property "class" of a tag element, from a CSS loaded into the HTML page.
    $("tag").removeClass("className") - This removes a class Name from the property "class" of the object
    $("tag").toggleClass("className") - This removes or adds a class Name from the property "class" of the object

Fetching text from HTML tag

  • $("#ID_NAME").text() - Fetches text from the tag with ID
    $("#ID_NAME").text("new text") - Changes text from the tag with ID
  • $("#ID_NAME").html() - Fetches html content from the tag with ID
    $("#ID_NAME").html("<>") - Changes html content from the tag with ID
